Oi! Vi que você está implementando uma árvore binária de busca. Qual operação está te dando problema? 🌳
A rotação AVL — não entendo quando aplicar rotação dupla...
Faz sentido! Rotação dupla acontece quando o filho e o neto estão em lados opostos. Olha: se o nó está desbalanceado à esquerda mas o filho está à direita, primeiro você faz uma rotação esquerda no filho, depois rotação direita no nó raiz.

FAQ

Perguntas frequentes.

Preciso saber matemática avançada para entrar?+
Não! O curso começa do zero. Toda a matemática e estruturas de dados são ensinadas de forma aplicada e progressiva dentro do currículo.
Quanto tempo preciso por semana?+
Sábado (2–3h) e domingo (2–4h). Recomendamos 2–3h de prática extra na semana. AI Tutor disponível 24/7 para dúvidas.
É 100% online?+
Sim, totalmente online e ao vivo. Gravações disponíveis para revisão a qualquer hora.
Vou aprender C++ sendo iniciante?+
C++ é introduzido nos meses 5–6, quando você já domina Python, estruturas de dados e algoritmos. A transição é gradual e guiada.
Vou sair com portfólio?+
Sim! Pelo menos 6 projetos reais no GitHub — incluindo sistemas com estruturas de dados implementadas do zero e um motor de busca em C++.
